Situs menunjuk ke halaman tertentu dan aplikasi web lebih merupakan semacam portal untuk konten dan informasi tertentu. Situs web tidak melakukan interaksi dua arah, sedangkan aplikasi web melakukan interaksi dua arah. Website adalah kumpulan suatu halaman di dalam suatu domain di internet yang dapat di akses melalui browser dengan tujuan menyampaikan suatu informasi kepada pengguna. Sedangkan aplikasi web memungkinkan interaksi didalamnya, misalnya mengirim formulir, menggunakan fitur chat, pembayaran online, dan lain-lain. Perbedaan pada tugas utama kalau website memberikan informasi kepada pembaca, sedangkan web app pengolahan data dan informasi secara otomatis.

Front end adalah bagian yang langsung dilihat dilihat oleh user dari website, webpage, or app. User juga bisa langsung berinteraksi pada bagian ini. Bagian ini dibangun menggunakan HTML, CSS, dan Java Script. Back End adalah bagian belakang layar dari sebuah website. Bahasa pemograman untuk back end development diantaranya adalah PHP, Ruby, Python, dan banyak lainnya. Full Stack developer bekerja pada bagian front end dan back end. Mereka menguasai HTML, CSS, Java Script, dan satu atau lebih bahasa pemograman back end. Bahasa pemrograman dalam back end biasanya menggunakan PHP, RUby, Python.
